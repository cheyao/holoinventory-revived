# Holoinventory Revived

A port of the 1.12.1 HoloInventory mod to newer Minecraft versions.

Ever wanted to have a simple way to see the contents of chests/machines without opening them? Tired of opening tons of chests to find the exact item you want? Now with this mod you can have a holographic display of items in storage containers!

![Holographic view over chest](https://files.catbox.moe/matqyy.png)

The mod not only works with vanilla Minecraft containers, but also modded ones! Preview the contents and outputs of your machines without opening them.

![Holographic view over modded blocks](https://files.catbox.moe/8s41zl.png)

### Usage

To use, install this mod and join a world. Now when looking at a container you can press **shift or control** to make a holographic view pop-up.

To customize the mod's behavior, go to the mod config menu and check the option buttons. You can enable **always active** in the config menu and disable keys. ([ModMenu](https://modrinth.com/mod/modmenu/) needed on Fabric. Or edit config file at `config/holoinventory-revived.toml`)

Futhermore, you can craft the holo glasses item and equip it to constantly have a holographic view:

![Crafting recipie](https://files.catbox.moe/wllp54.png)

(For versions where the spyglass doesn't exist, use a glass panel instead)

### Misc

This mod needs to be installed both on the client and server side to work. It implements caching so it won't take up much serverside resources.

Credit for the pre-1.12.1 holoinventory mod goes to [Dries007](https://www.curseforge.com/minecraft/mc-mods/holoinventory)

Need help/found a bug? Open an issue on the [GitHub repo](https://github.com/cheyao/holoinventory-revived/issues).
